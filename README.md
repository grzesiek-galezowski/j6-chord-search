# j6-chord-search
A WPF app for Roland J-6 to take a series of chords and display all scales that have these chords (including transposed)

## Web Version
The web-based J-6 Chord Browser is available online at: https://grzesiek-galezowski.github.io/j6-chord-search/

You can also find the web version in the `web/` directory.
